## 介绍

一款自用的物资管理系统

## 安装

构建镜像

```bash
docker build -t material-manager:0.0.8 .
```

启动容器

```bash
docker run -d --restart=always --name=material-manager -p 8080:80 material-manager:0.0.2
```

访问 http://localhost:8080/ 即可
